# flutterLoadingView
![Image text](https://github.com/475789729/flutterLoadingView/blob/master/test.gif)

<br>使用说明:<br>
复制lib/loadingview文件夹到您的项目里面，其中每一个文件对应一种loadingview<br>
第一行从左到右是:<br>
BallScaleMultipleIndicator, BallScaleIndicator, BallPulseIndicator<br>
第二行从左到右是:<br>
BallSpinFadeLoaderIndicator, MaterialProgressIndicator, SquareSpinIndicator<br>
